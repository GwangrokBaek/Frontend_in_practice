# minify
CSS 는 네트워크를 통해서 전송되기에 CSS 의 크기가 커지면 컨텐츠의 생산자와 소비자 모두에게 손해이다.
이때 코드의 크기를 줄이는 것을 통해서 이런 문제를 완화해주는 도구가 minify 도구이다.

실제로 동작되지 않는 쓸모없는 디자인을 제거하거나 줄바꿈 및 공백을 제거하는 식으로 크기를 줄인다.
